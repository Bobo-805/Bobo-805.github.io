git 操作指南

git

安装

  • Linux

    • apt-get install git-all
    • yum install git-all

  • Win

    • 下载地址

管理

  • 创建版本库

    • git config --global user.name "bobo"
    • git config --global user.email "bobo@123.com"
    • git init

  • 添加文件

    • git add [文件]
      • 文件可写可不写， "."表示全部文件

  • 查看状态

    • git status
    • git status -s
      • "-s": status 的缩写模式

  • 提交改变

    • git commit -m "log信息"

  • git 流程

记录修改

  • 修改记录

    • git log
    • git log --oneline
      • "--oneline": 每个 commit 内容显示在一行
    • git reflog
      • 最近所有修改的记录

  • 查看修改

    • git diff (未add部分与commit部分)
      • "-" 表示删去的内容
      • "+" 表示增加的内容
    • git diff --cached (add部分与commit部分)
    • git diff HEAD (未add部分与add部分)

  • 修改历史

    • 修改已 commit 的版本
      • git commit --amend --no-edit
        • "--no-edit": 不编辑, 直接合并到上一个 commit

  • 还原版本

    • 回到 add 之前
      • git reset 文件
    • 回到 commit 之前
      • git reset --hard HEAD
    • 回到上一次 commit
      • git reset --hard HEAD^
        • 有多少 ^ 代表回到多少次之前
        • ^ 也可以用 ~n 表示
    • 回到指定的commit
      • git reset --hard id
        • 可以通过git log --oneline查看之前的 id

  • 还原文件

    • git checkout id -- 文件
